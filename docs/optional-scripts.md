# Optional / advanced scripts

These scripts remain in `scripts/` for power users but are not part of the main showcase path:

| Script | Purpose |
|--------|---------|
| `scripts/pack-team-zip.ps1` | Bundle a zip for offline distribution |
| `scripts/bench-models.mjs` | Rough latency comparison across models |
| `scripts/probe-llm.mjs` / `probe-chat-completions.mjs` | Raw API connectivity probes |
| `scripts/graph-login.ts` / `run-pipeline.ts` | Microsoft Graph login + full pipeline (tenant permitting) |

Day-to-day: use `npm run demo`, `npm run classify`, `npm test`, and the Outlook macros in `outlook/`.
